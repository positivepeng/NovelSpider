三、字体设置

38402019分辨率： body{ font-size: 64px; } 19201080分辨率： body{ font-size: 32px; } 子盒子以body字体为基数 例： div{ font-size: px2em(24,64); } 注： 1.分辨率变了，div的字号不用变，它是body的倍数，只需更改不同分辨率下body的字号即可
